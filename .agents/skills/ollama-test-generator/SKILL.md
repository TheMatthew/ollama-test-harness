# Ollama Test Generator

This skill guides the user (and Agents) in generating new benchmarks/tests for the `ollama-test-harness`.

## Workflow

### 1. Scaffold the New Task
You can use the bundled python script to scaffold a new task directory containing both the prompt Markdown template and the JSON configuration file:

```bash
/home/matthew/.gemini/skills/ollama-test-generator/scripts/generate_test.py --name <task_name> --language <language>
```

For example, to create a new task named `fibonacci` for `python`:
```bash
.agent/skills/ollama-test-generator/scripts/generate_test.py --name fibonacci --language python
```

This creates:
- `tasks/fibonacci/fibonacci_python.json`
- `tasks/fibonacci/fibonacci_python.prompt.md`

### 2. Configure the Prompt File
Edit the generated `tasks/<task_name>/<task_name>_<language>.prompt.md` file to specify the requirements of the code to be generated by the model. 

### 3. Add Validation Rules
Edit the generated `tasks/<task_name>/<task_name>_<language>.json` file. Set up appropriate `build_command` and `run_command` templates, and define validation rules under the `"validation"` array.

Supported rule types in `ollama-test.py`:
- **`line_count`**:
  ```json
  { "type": "line_count", "expected": 32 }
  ```
- **`line_width`**:
  ```json
  { "type": "line_width", "width": 64 }
  ```
- **`single_char_at`**:
  ```json
  { "type": "single_char_at", "line": 0, "index": 32, "char": "#", "label": "Generation 0" }
  ```
- **`symmetric_pair`**:
  ```json
  { "type": "symmetric_pair", "line": 1, "indices": [31, 33], "char": "#", "label": "Generation 1" }
  ```

### 4. Run the Test
To execute the test benchmark:
```bash
python3 ollama-test.py --task tasks/<task_name>/<task_name>_<language>.json
```
